# Image Parsing and Generation Workflow

Use this reference for competitor image analysis, product image planning, and image-generation prompts.

## Competitor Image Parsing Template

For each image, capture:

- Image role: main image, lifestyle, feature callout, size comparison, package contents, review/UGC, ad creative.
- Composition: camera angle, crop, background, lighting, props, negative space.
- Message hierarchy: headline, subclaims, icons, arrows, comparison blocks, labels.
- Product evidence: visible materials, dimensions, texture, accessories, packaging.
- Conversion purpose: trust, clarity, desire, urgency, objection handling.
- Compliance/IP risk: copied layout, competitor logo, unverifiable badge, medical/safety claim, misleading scale.

## Safe Strategy Extraction

Do:
- Extract reusable ideas such as `show scale in hand`, `show before/after organization concept without false claims`, or `show package contents clearly`.
- Create original layouts, colors, props, and wording aligned to the user's brand.
- Mention when real product photography is required.

Do not:
- Copy a competitor image, exact layout, logo, slogan, watermark, model likeness, or branded packaging.
- Invent certifications, lab results, awards, review stars, or platform badges.
- Create misleading product dimensions, effects, or impossible use cases.

## Prompt Blocks

### Hero Image Prompt

Include:
- product description, color/material, exact visible components;
- clean background requirement;
- camera angle and lighting;
- no text unless platform permits;
- ratio, resolution, and margin guidance.

### Lifestyle Prompt

Include:
- target customer and scene;
- product in realistic use;
- emotional benefit;
- props that support but do not distract;
- region/season if relevant.

### Feature Callout Prompt

Include:
- one feature per image;
- simple headline text if supported;
- arrows/icons only when needed;
- factual dimensions/specs from user data;
- brand color palette.

### Negative Prompt

Always include:
`no competitor logo, no fake certification, no false review stars, no unreadable text, no distorted product, no extra accessories not included, no medical or guaranteed performance claims, no watermark`.

## Image Set Recommendation

Minimum launch pack:

1. Main image on compliant clean background.
2. Lifestyle image for the primary use case.
3. Feature callout image for the top differentiator.
4. Size or capacity comparison.
5. Package contents image.
6. Usage steps or care instructions.
7. Brand/trust/service image if claims are true.

## QA Checklist

- Product appearance matches the real SKU and variant.
- Text is readable at mobile size.
- Claims match listing copy and available proof.
- No competitor IP appears.
- No fake badges, certifications, reviews, or platform UI.
- Image dimensions and background meet platform requirements.
