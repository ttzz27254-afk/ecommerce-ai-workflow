# Marketplace Field Map

Use this reference when creating listing/upload sheets. Adjust field names to the user's exact platform template when they provide one.

## Universal Product Fields

| Field | Purpose | Operator guidance |
| --- | --- | --- |
| SKU | Internal stock identifier | Keep unique; use brand-category-variant sequence if the user has no SKU system. |
| Product title | Search and conversion | Put primary keyword early, include brand only if useful, avoid unsupported claims. |
| Brand | Brand registry/store identity | Do not invent brand authorization. |
| Category | Platform browse placement | Choose the closest category; flag uncertain categories. |
| Variant | Color/size/style grouping | Keep variant labels customer-friendly and consistent. |
| Price | Listing price | Compare with competitor price band before finalizing. |
| Inventory | Initial available stock | Keep consistent with fulfillment system. |
| Main image | Primary search image | Usually product-only, clear background; obey platform image rules. |
| Bullet points | Conversion copy | Benefit first, then feature/proof/use case. |
| Description | Detail page content | Explain scenarios, package contents, care instructions, and guarantee if true. |
| Search terms/tags | SEO discovery | Use relevant synonyms; avoid competitor trademarks unless authorized. |
| Compliance notes | Risk control | Record claims that need proof, certificates, or legal review. |

## Copy Formulas

### SEO Title

`Brand + Core Keyword + Key Differentiator + Use Case + Variant/Size`

Rules:
- Keep the first 60-80 characters highly relevant.
- Avoid keyword stuffing and repeated phrases.
- Do not use `best`, `No.1`, `official`, `medical-grade`, `FDA-approved`, or similar claims unless proven and platform-compliant.

### Five Bullet Points

1. **Benefit**: What problem it solves.
2. **Differentiator**: Why this product instead of competitors.
3. **Scenario**: Where/how customers use it.
4. **Proof/detail**: Materials, dimensions, included parts, compatibility.
5. **Risk reduction**: Care, service, warranty, or clear limitation if true.

### Description

Recommended sections:
- One-sentence positioning.
- 3-5 use scenarios.
- Specifications and package contents.
- Care/safety notes.
- Brand/service promise if true.

## Platform Notes

- **Amazon**: prioritize title/bullets/search terms; be strict with claims and main-image rules.
- **Shopify**: include SEO title/meta description, collection tags, alt text, and rich description blocks.
- **TikTok Shop**: keep copy simpler; emphasize video hooks, proof points, shipping, and creator-friendly claims.
- **Shopee/Lazada**: include localized keywords, promo mechanics, shipping dimensions, and chat/service promises.
- **Etsy**: emphasize handmade/vintage/supply eligibility, tags, materials, occasion, and personalization.
